# 项目简介

## 核心功能
- 提供基于YOLOv8的腐蚀检测系统完整源码
- 包含600张腐蚀数据集，支持['腐蚀']单类检测
- 已训练好模型，配置成功环境可直接使用
- 支持图片、视频及摄像头进行检测
- 支持选择模型，自定义置信度和IOU阈值
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持选择显示标签和原图

## 适用场景
- 金属结构腐蚀检测
- 工业设备维护检查
- 材料老化评估
- 质量控制系统
- 安全监测应用

## 技术特点
- 开发环境：Pycharm+Anaconda或VSCode+Anaconda
- 技术栈：python3.9、opencv-python、PyQt5、ultralytics、torch1.9等
- 系统文件包含完整程序文件、UI界面源文件、图标、数据集图片和项目运行教程
- 提供图形化界面，操作便捷
